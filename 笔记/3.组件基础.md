# 什么是组件

组件就是把很多的标签组合成一个 功能模块

为什么要使用组件：

就是为了： 复用 易维护


# 创建组件

Vue.component(name: string, options: object)


1. 组件作用域是在当前实例中的

组件只能使用自己选项里的东西

父级的不能使用 子级的也不能使用


> 在html 里面去写 组件的时候 必须使用双tag
> template 选项中必须只能有一个根tag
> template 选项会替换掉挂载的元素
> 组件名称 大驼峰命名 在使用时可以小驼峰使用
> 组件名称 小驼峰命名 在使用时不可以大驼峰


### options

1. template
2. props

是用来获取组件的`挂载元素（组件名称）` 属性
